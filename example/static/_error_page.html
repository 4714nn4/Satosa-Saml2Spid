AuthProxy Error Page

  • Authentication Proxy Error Page

    Dear User,

    There has been an exception in managing your Authentication flow. This cannot depend on your credentials, otherwise it would have been the page where you entered these to provide you with a Failed Authentication notice.

    The problem you have encountered has been communicated in real time to the platform administrators, in the event that it was a temporary problem we ask you to:

    - Try to authenticate again, starting from the page of the service provider website;
    - If the problem occurs again, wait a few minutes and try again;
    - If the problem persists for more than 24 hours, we ask you to open a support ticket at https://ticket.unical.it.

    In most cases, it is a temporary problem that arose following an update of the platforms in progress, an advancement of the software versions or a server migration.
